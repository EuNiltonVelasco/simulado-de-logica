# 🧠 Simulado V ou F - Quiz Interativo

Projeto de um **simulado interativo de Verdadeiro ou Falso**, desenvolvido com HTML, CSS e JavaScript puro, focado em prática para provas e reforço de aprendizado.

---

## 🚀 Funcionalidades

* ✅ Quiz com 30 perguntas (V ou F)
* 📊 Barra de progresso em tempo real
* 🔒 Bloqueio de respostas após seleção
* 💬 Feedback imediato (acertou/errou)
* 📈 Cálculo automático da nota final (%)
* 🎨 Feedback visual por desempenho:

  * 🟢 70% ou mais → Bom desempenho
  * 🔵 50% a 69% → Pode melhorar
  * 🔴 Abaixo de 50% → Atenção
* 🏆 Mensagem especial ao gabaritar (100%)
* 🎉 Efeito de confete ao acertar tudo
* 💾 Salvamento automático do progresso (localStorage)
* 🔄 Recuperação do estado após atualizar a página (F5)

---

* O progresso é salvo automaticamente no navegador (localStorage)
* Mesmo atualizando a página, o estado do simulado é mantido
* Para reiniciar o simulado, limpe o cache/localStorage do navegador
