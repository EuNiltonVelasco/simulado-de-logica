1. Variável é um espaço reservado na memória do computador para armazenar valores que podem ser manipulados ao longo da execução de um programa.

3. A estrutura básica de um algoritmo é composta por entrada, saída e processamento de dados.

4. Uma variável em programação é um espaço na memória que pode armazenar valores que podem ser alterados durante a execução do programa, enquanto uma constante é um espaço na memória que armazena um valor fixo, que não pode ser modificado após a sua definição.

5. Um algoritmo é uma sequência de passos ou instruções que são seguidos para resolver um problema ou realizar uma tarefa específica. Em programação, um algoritmo define a lógica para que o computador realize uma tarefa, como fazer cálculos, organizar dados ou interagir com o usuário.

8. Operadores relacionais são, respectivamente: == (igual), <> (diferente), >= (maior ou igual).

9. Para exibir uma mensagem ao usuário, usamos o comando escreva(mensagem).

11. No comando condicional se...senão, o senão é sempre executado quando a condição do se for verdadeira.

12. Toda vez que abrimos uma condição “se”, precisamos fechá-la com “fimse” (em VisualG).

14. A estrutura condicional se...então...senão é utilizada dentro de um algoritmo para executar diferentes blocos de código dependendo de uma condição.

15. A maneira correta de escrever um se de múltiplas condições no VisualG é: Se (x > 5 e x < 10) então.

17. Em VisualG, quando uma condição se for falsa, a execução seguirá para o bloco senão.

19. Em uma estrutura se do VisualG, o fimse não é necessário para fechar a estrutura condicional se não houver um senão.
